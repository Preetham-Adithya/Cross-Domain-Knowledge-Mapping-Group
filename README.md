Mental Health Dashboard

Project Overview :


The Mental Health Dashboard is a web-based application built using Streamlit for analyzing and visualizing mental health data. It allows users to upload custom datasets, explore insights via interactive graphs, submit feedback, and use semantic search. An admin panel is included to manage feedback and oversee the platform.

This project is intended to help researchers, practitioners, and organizations make data-driven decisions related to mental health.

Features :


User Authentication: Login system for standard users and admin.


Admin Panel: Admin-only access to view feedback and manage submissions.


Dataset Upload: Users can upload their own CSV datasets for analysis.


Feedback System: Users can submit feedback which is saved and visible to admin.


Data Visualization: Interactive charts and graphs generated from uploaded datasets.


Semantic Search: NLP-based search for meaningful insights.


Responsive UI: Clean interface powered by Streamlit components.
